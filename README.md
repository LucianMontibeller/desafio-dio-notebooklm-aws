## 🤖 Engenharia de Prompts e "Cicatrizes"
Neste projeto, apliquei técnicas de engenharia de prompts para transformar documentação técnica em conhecimento prático.

### Prompt Estratégico (Sucesso)
> *"Explique a diferença entre Elasticidade e Escalabilidade na AWS usando uma analogia com uma pizzaria que precisa atender mais clientes em um sábado à noite."*

* **Resultado:** A IA comparou a Escalabilidade ao aumento do número de fornos e a Elasticidade à contratação temporária de entregadores apenas para o horário de pico.
* **Aprendizado:** O uso de analogias do cotidiano facilita a retenção de conceitos complexos de infraestrutura.

### "Cicatrizes" (Dificuldades e Troubleshooting)
* **Desafio:** Solicitações amplas como "resuma a AWS" geraram respostas genéricas e pouco úteis.
* **Solução:** Fragmentei as perguntas por pilares (Segurança, Armazenamento e Computação).
* **Lição:** O mercado valoriza a capacidade de "saber perguntar" (prompting) de forma granular para obter respostas acionáveis.

## 📝 Miniguia de Estudo (Entrega Final)

### 1. Resumo Estruturado: O que é Cloud Computing?
A computação em nuvem é a entrega de recursos de TI sob demanda via internet. Em vez de gerenciar hardware físico, utilizamos serviços gerenciados:
* **EC2:** Poder de processamento (servidores).
* **S3:** Armazenamento de objetos com alta durabilidade.
* **RDS:** Bancos de dados relacionais simplificados.
* **IAM:** Gestão de identidades e permissões de segurança.

### 2. Glossário de Conceitos Chave
* **Alta Disponibilidade:** Sistemas que operam com o mínimo de interrupção possível.
* **Pay-as-you-go:** Modelo financeiro onde paga-se apenas pelo recurso consumido.
* **Zonas de Disponibilidade (AZs):** Data centers isolados dentro de uma Região para garantir redundância.
